Upload the app screenshots here with these exact names — the cards on the
site pick them up automatically (any missing file simply falls back to the
logo watermark, nothing breaks):

| File | Card |
|---|---|
| `data-server.png` | Data Server |
| `meetsense.png` | MeetSense |

Recommended: landscape screenshots (≈16:9, e.g. 1280×720 or wider). They are
cropped with `object-fit: cover` anchored to the top, and a dark gradient is
laid over the lower half so the card title stays readable.
